Machine Learning Projects

Audio Fingerprinting Project

The "Audio Fingerprinting" project is a machine learning endeavor that focuses on audio classification. Its primary goal is to accurately categorize audio samples into distinct music genres through the implementation of a deep learning model. The project encompasses several key components, starting with data preparation, including setting the audio parameters and loading audio files using the Librosa library.

Feature extraction is a crucial step, which involves creating a feature matrix 'X' to represent audio characteristics. The training dataset comprises various music genres organized in subdirectories, with features extracted from each audio sample. A neural network model is constructed using TensorFlow and Keras, consisting of an input layer, hidden layers with ReLU activation, dropout layers to prevent overfitting, and an output layer for multi-class classification. Model compilation is accomplished with the Adam optimizer and sparse categorical cross-entropy loss.

The model is trained using the feature matrix and labels from the training dataset, and the evaluation metric chosen is accuracy. A list of music genre labels is provided to map label indices, and the model's performance is evaluated by predicting the music genre of a test audio sample. This project represents a comprehensive approach to audio data processing, feature extraction, deep learning, and multi-class music genre classification.

Facial Landmark Prediction Project

The "Facial Landmark Prediction" project is a computer vision and deep learning endeavor with the aim of detecting and predicting facial landmarks in images. Facial landmarks are crucial points on a person's face, encompassing features such as the eyes, nose, and mouth.

This project involves data preparation, face detection, and the extraction of 68 facial landmarks from detected faces using the dlib library. The data is organized for model training and testing, and a Convolutional Neural Network (CNN) model is constructed to predict these facial landmarks.

Once trained, the model can make predictions on test images, and the predicted landmarks are overlaid on the images for visualization. This project provides a valuable tool for tasks like facial recognition, emotion analysis, and other computer vision applications, making it a noteworthy addition to your portfolio.
